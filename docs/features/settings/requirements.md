# 設定 要件

## 概要

顧客プロファイルと端末マスタを管理する機能。
他のすべての機能（採取・閲覧・評価）が使う設定の基盤となる。

## 顧客プロファイル

1つの「現場」に対応する設定セット。ログイン認証ではなく、設定の切り替え単位。

| 項目 | 説明 |
|------|------|
| プロファイル名 | 識別用の名前（例: "案件A 1次試験"） |
| アーカイバパス | エントリ解凍用アーカイバの実行ファイルパス |
| 端末マスタ | 接続先端末の一覧 |
| 採取設定 | `CollectionTarget` の定義（採取要件参照） |

詳細スキーマ: [docs/schema/config.md](../../schema/config.md)

## 端末マスタ

採取・閲覧の対象となるリモート端末の接続情報。

| 項目 | 説明 | 備考 |
|------|------|------|
| IPアドレス | 端末のIP（例: 192.168.1.10） | 識別キー |
| 表示名 | UI上の識別用ラベル | 任意 |
| 閲覧設定 | 閲覧時のパス・フォーマット設定 | 詳細は未確定 |
| 採取設定 | `CollectionTarget` の参照 | |

## ユースケース

| # | 操作 | 説明 |
|---|------|------|
| 1 | プロファイル一覧表示 | 登録済みプロファイルの一覧を表示 |
| 2 | プロファイル切り替え | 使用するプロファイルを選択する |
| 3 | 端末マスタ編集 | IP・表示名・設定の追加・変更・削除 |
| 4 | 採取設定編集 | `CollectionTarget` の追加・変更・削除 |
